Getting Started

Requirements

  • Mac OS X
  • Composer
  • PHP 5.6+
  • Laravel Valet (Optional, allows for automatically setting up .dev domains)

Install

  1. Install Composer if not already installed.
    • Installation instructions
    • Make sure the ~/.composer/vendor/bin directory is in your system's "PATH".
  2. Install PHP if not already installed, you can use Homebrew to easily install PHP.
    • Install Hombrew or if Homebrew was already installed make sure it is up to date via brew update.
    • Install PHP 7.0 via brew install homebrew/php/php70.
  3. Install Press CLI via composer global require dholloran/press-cli.
  4. (Optional) Install Laravel Valet
